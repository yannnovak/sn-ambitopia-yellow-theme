# Ambitopia \[yellow\] Theme

A dark [Standard Notes](https://standardnotes.org/) theme from the Ambitopia design system (yellow variant).

**Other variants:** [Red version](https://github.com/yannnovak/sn-ambitopia-red-theme)

## Preview

<img src="preview.png"/>

## Installation

Open "Plugin" in Standard Notes Preferences and enter the following URL under "Install Custom Plugin", then click install.

```
https://cdn.jsdelivr.net/gh/yannnovak/sn-ambitopia-yellow-theme@main/ext.json
```
